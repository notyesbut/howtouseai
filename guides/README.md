# Guide Library

This folder is the growing guide library for `howtouseai`.

The goal is to turn the repository into a clear, beginner-friendly knowledge base for:

- AI coding
- AI agents
- prompting and workflow
- safe Git and debugging habits
- self-hosting and deployment
- infrastructure basics
- RAG and retrieval workflows
- embeddings and vector search
- recommendation systems
- stack selection and practical tradeoffs
