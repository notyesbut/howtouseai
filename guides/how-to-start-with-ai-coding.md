# How to Start with AI Coding

Welcome. If you are new to AI coding, this is your starting point.

You do not need to know everything in advance. You do not need to already be an engineer. You just need to choose the right setup path and follow it step by step.

## The Goal

The goal is simple:

**Pick your path -> set up your environment -> connect your coding tools -> start building.**

## What Is AI Coding?

AI coding means using coding agents and AI-assisted tools to help you:

- write code
- fix bugs
- explain code
- build websites
- build apps
- automate tasks
- launch projects faster

You are not "installing AI itself."

You are setting up an AI coding environment so you can work with coding agents on your computer or server.

## Start Here

Answer this question first:

**Do you have a computer or laptop?**

## Quick Route Map

Use this simple decision guide:

1. If you have a computer and want the easiest starting point, begin with a local setup.
2. If you have a computer but want a cleaner or more powerful setup, use a remote server.
3. If you do not have a computer, start with a phone-first remote server setup.

## Path 1: Yes, I Have a Computer

If the answer is yes, the next question is:

**Do you want to run AI coding locally on your own device?**

### Option A: Yes, locally

This is usually the easiest way to start if you already have a laptop or PC.

Best for:

- beginners
- people who want the simplest first setup
- people who want to learn step by step on their own machine

Typical local paths:

#### Windows

`Windows -> WSL2 -> Ubuntu -> Node.js -> Git -> VS Code -> coding agent`

#### macOS

`macOS -> Homebrew -> Node.js -> Git -> VS Code -> coding agent`

#### Linux

`Linux -> base packages -> Node.js -> Git -> VS Code or terminal -> coding agent`

### Option B: No, not locally

If you do not want to run everything on your own computer, you can use a remote server instead.

Best for:

- weaker laptops
- people who want a cleaner remote setup
- long-running tasks
- users who want to connect from different devices

Typical server path:

1. Rent a VPS.
2. Choose Ubuntu 24.04.
3. Connect with SSH.
4. Install Node.js.
5. Install Git.
6. Install your coding agent.
7. Work remotely.

## Path 2: No, I Do Not Have a Computer

You can still start.

If you have:

- a phone
- internet access
- the ability to rent a small server

then you can still use AI coding through a remote Ubuntu server.

This is the phone-first path:

1. Rent a VPS.
2. Connect through SSH.
3. Use a terminal app like Termius.
4. Work on the remote machine.
5. Run your coding tools there.

So no, a laptop is not strictly required to begin.

## Quick Decision Guide

Choose the path that fits you best:

- If you already have a laptop or PC and want the simplest start, begin locally.
- If your device is weak or you want a cleaner setup, use a remote server.
- If you only have a phone, start with a VPS and SSH.

## Final Note

The hardest part for most people is not learning everything. It is simply choosing a path and starting.

Start with the setup that is available to you today. You can always improve it later.
